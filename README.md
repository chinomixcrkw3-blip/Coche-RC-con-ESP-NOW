En el archivo PDF te dejo el circuito tanto para el coche como para el control, revisa las conexiones cuando hagas tu circuito.
El codigo y el circuio estan hechos con un ESP32-S3, si vas a utilizar otra placa revisa que pines tiene disponibles y cuales vas a usar,
modifica los pines en el circuito y el código.
Si tienes problemas al crear el proyecto con estos archivos, puedes crear un proyecto nuevo y solo copia y pega el contenido de los archivos
de la carpeta main.
En el codigo para el control agrega la direccion MAC del ESP32 del coche, la encuentras abreiendo el monitor serial, si no te aparece abre
el monitor serial y reinicia el ESP32.
